## 设计目标

### 核心问题
- TUI 模式存在许多 Bug
- Agent 使用方式不便利
- 复杂的命令系统用户难以记忆
- 许多功能尚未实现

### 设计原则
1. **从命令驱动转向面板驱动** - 可视化操作，无需记忆命令
2. **上下文相关交互** - 根据当前面板显示可用操作
3. **单键操作** - 常用功能只需按一个键
4. **即时反馈** - 所有操作都有视觉反馈

## 整体架构

### 布局设计

```
┌─────────────────────────────────────────────────────────────┐
│  DevBoost v0.2.0                   [Agent: ●运行] [模型:Sonnet] │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                   │
│  1.对话   │  主面板区域（动态内容）                             │
│  2.Agent │  - 对话模式：显示对话历史                           │
│  3.模型   │  - Agent模式：显示Agent状态和操作                   │
│  4.工具   │  - 模型模式：显示模型列表                           │
│  5.设置   │                                                   │
│          │                                                   │
├──────────┴──────────────────────────────────────────────────┤
│ > 输入消息...                       按 ? 查看快捷键            │
└─────────────────────────────────────────────────────────────┘
```

### 关键决策

1. **命令系统**: 保留部分基本命令（`/model`、`/skills` 等），参考其他项目实现
2. **简单模式**: 保留 `--no-tui` 仅用于开发者测试，正式安装包不提供此选项
3. **向后兼容**: 自动迁移旧配置，不强制用户手动更新

## 面板系统

### 1. 对话面板 (快捷键 1)

**功能**:
- 显示对话历史（自动保存和加载）
- 支持搜索历史记录（按 `/`）
- 对话导出

**快捷键**:
| 快捷键 | 功能 |
|--------|------|
| `c` | 清空对话 |
| `s` | 保存对话到文件 |
| `l` | 加载历史对话 |
| `e` | 导出对话 |
| `/` | 搜索历史 |

### 2. Agent 面板 (快捷键 2)

**功能**:
- 实时显示 Agent 状态
- 可视化运行状态指示
- Agent 日志查看

**状态指示**:
- `● 运行中` - 绿色
- `○ 已停止` - 灰色
- `⚠ 错误` - 红色
- `⏳ 启动中` - 黄色

**快捷键**:
| 快捷键 | 功能 |
|--------|------|
| `Enter` | 启动/停止 Agent |
| `r` | 重启 Agent |
| `i` | 查看详细信息 |
| `l` | 查看 Agent 日志 |

### 3. 模型面板 (快捷键 3)

**功能**:
- 显示所有已配置模型
- 当前模型高亮显示
- 添加/删除/编辑模型

**快捷键**:
| 快捷键 | 功能 |
|--------|------|
| `Enter` | 切换到选中模型 |
| `a` | 添加新模型（向导式） |
| `d` | 删除选中模型 |
| `e` | 编辑模型配置 |

### 4. 工具面板 (快捷键 4)

**功能**:
- 显示所有可用工具
- 按类别分组（内置、自定义）
- 工具启用/禁用切换

**快捷键**:
| 快捷键 | 功能 |
|--------|------|
| `Enter` | 查看工具详情 |
| `t` | 切换工具启用状态 |
| `+` | 添加自定义工具 |

### 5. 设置面板 (快捷键 5)

**功能**:
- 配置管理
- Telegram 集成设置
- 主题切换

**快捷键**:
| 快捷键 | 功能 |
|--------|------|
| `e` | 编辑配置 |
| `r` | 重置为默认 |
| `t` | Telegram 设置 |
| `T` | 切换主题 |

## 快捷键系统

### 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `1-5` | 切换面板 | 数字键直接跳转 |
| `q` / `Esc` | 返回/退出 | 返回上级或退出 |
| `?` / `x` | 选项菜单 | 显示当前可用操作 |
| `Ctrl+C` | 强制退出 | 紧急退出 |

### 导航快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `↑` `↓` 或 `j` `k` | 上下移动 | 选择列表项 |
| `Enter` | 确认选择 | 执行选中项 |
| `g` `G` | 跳转 | 跳到顶部/底部 |
| `/` | 搜索 | 在列表中搜索 |

### 选项菜单示例

按 `x` 或 `?` 显示动态菜单：

```
┌─────────────────────────────────────┐
│              可用操作                 │
├─────────────────────────────────────┤
│  Enter  启动/停止 Agent              │
│  r      重启 Agent                   │
│  i      查看详细信息                 │
│  l      查看日志                     │
│  ────────────────────────────────   │
│  ?      显示帮助                     │
│  q      返回                         │
└─────────────────────────────────────┘
```

## 组件架构

### 目录结构

```
packages/tui/src/
├── core/
│   ├── App.ts                 # 主应用控制器
│   ├── ScreenManager.ts       # 屏幕管理
│   └── EventManager.ts        # 事件处理
├── panels/
│   ├── BasePanel.ts           # 面板基类
│   ├── ChatPanel.ts           # 对话面板
│   ├── AgentPanel.ts          # Agent 面板
│   ├── ModelPanel.ts          # 模型面板
│   ├── ToolsPanel.ts          # 工具面板
│   └── SettingsPanel.ts       # 设置面板
├── menus/
│   ├── OptionsMenu.ts         # 选项菜单
│   ├── Wizard.ts              # 向导基类
│   └── AddModelWizard.ts      # 添加模型向导
├── components/
│   ├── StatusBar.ts           # 状态栏组件
│   ├── InputBox.ts            # 输入框组件
│   ├── Notification.ts        # 通知组件
│   └── ThemeManager.ts        # 主题管理
└── layouts/
    └── MainLayout.ts          # 主布局
```

### 面板基类

```typescript
abstract class BasePanel {
  abstract id: string;
  abstract title: string;
  abstract keybindings: KeyBinding[];

  abstract render(): void;
  abstract handleInput(key: string): void;

  onFocus(): void;
  onBlur(): void;
  refresh(): void;
}
```

### 数据流

```
用户操作 → Panel → Event → App → Config → Render
                ↑                   ↓
                └───────────────────┘
                    状态更新通知
```
